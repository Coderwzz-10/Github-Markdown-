3、**文本块**

（1）常用文本块语法：使用反引号`，在输入文本的开头和结尾各连续三个。

\`\`\`<br>
一万年太久，只争朝夕<br>
\`\`\`

```
一万年太久，只争朝夕
```

（2）渲染代码块：在开头反引号后加上编程语言名称

例如：\`\`\`c++<br>
#include<iostream><br>
using namespace std;<br>
int main(){<br>
int a;<br>
a=1;<br>
cout<<a<<endl;<br>
}<br>
\`\`\`

```c++
#include<iostream>
using namespace std;
int main(){
int a;
a=1;
cout<<a<<endl;
}
```

~~文字~~
